```
echo "\"" -> Prints one " character
echo 'abc'\''abc' -> Prints abc'abc
echo "abc"\""abc" -> Prints abc"abc
echo 'abc'"'"'abc' -> Prints abc'abc
echo "abc"'"'"abc" -> Prints abc"abc
echo "Hello"', world!' -> Prints Hello, world!
echo "I like to use" '"double quotes"' "sometimes" -> Prints I like to use "double quotes" sometimes
```
